# White Rabbit - Client PRD

## 제품 정의

White Rabbit은 터미널에서 동작하는 1:1 세션 전용 P2P 통신 도구이다. 메시지는 서버와 로컬 어디에도 저장되지 않으며, 연결된 세션 동안에만 존재한다.

---

## 핵심 컨셉

- 메신저가 아닌 **전화/인터컴 모델**
- 온라인 상태에서만 연결 가능
- 세션 종료 시 모든 히스토리 폐기
- 키보드 전용 터미널 UX
- 사이버펑크 / 해커 감성 (매트릭스 모티브)

---

## 타겟 사용자

- 개발자, 보안/인프라 엔지니어
- 터미널/TUI 애호가
- 프라이버시를 중시하는 긱/해커 문화 사용자

---

## 플랫폼

- macOS
- Windows
- Linux (선택 지원)

---

## 라이선스

- 결제 후 이메일로 라이선스 키 지급
- 라이선스 키는 디바이스 1대에만 바인딩
- 최초 1회 활성화 시 서버 검증, 이후 오프라인 사용 가능

---

## 연락처 관리

- `rabbit add <code>` : 단방향 연락처 등록
- `pair` 방식으로 양방향 즉시 등록 가능
- 연락처 이름은 사용자가 로컬에서 직접 설정
- 이름은 언제든 수정 가능

---

## 연결 UX

- 상태: `OFFLINE` / `ONLINE` / `RINGING` / `CONNECTING` / `CONNECTED` / `BUSY`
- `CONNECTED` 상태는 항상 1명만 허용
- 통화 중 다른 요청은 자동 `BUSY` 응답
- `UNKNOWN` caller는 수락/거절 가능

---

## 세션 채팅

- 메시지 히스토리는 세션 동안만 유지
- 세션 종료 시 즉시 폐기
- 타이핑 효과로 한 글자씩 출력

---

## 키보드 UX

| 키 | 동작 |
|----|------|
| `↑` / `↓` | 연락처 선택 |
| `Enter` | 연결 / 수락 |
| `Esc` | 종료 / 거절 |
| `r` | 이름 변경 |
| `Ctrl+C` | 앱 종료 |

---

## 보안 원칙

- 메시지 저장 없음
- 로그 최소화
- 서버 신뢰 최소화
- Fingerprint 기반 식별

---

> *"The server remembers nothing. The client remembers only while connected."*
